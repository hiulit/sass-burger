Sass Vegan Burger

[Demo &raquo;](http://joren.co/sass-burger/)

## Install

Download the [_vegan-burger.scss file](_vegan-burger.scss)

## Usage

Take a look at the demo and the `_vegan-burger.scss` file, it should be pretty self-explanatory.
